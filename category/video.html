Live stream (онлайн стримы) - программное обеспечение

Sat 15 April 2017 | tags: linuxlivestreamobsvideostream

Цель данной страницы сохранить для быстрого развертывания live стрима видео на различные онлайн сервисы на система linux.

Одним из популярных программ это OBS Studio

На данный момент удалось запустить OBS Studio на Linux Mint 18.1.

На  Fedora 25 при запуске выдает ошибку графических драйверов: пока нет поддержки Wayland см https://www.linux.org.ru/forum/general/13199751

Linux Mint 18.1

sudo add-apt-repository ppa:obsproject/obs-studio
sudo apt-get update
sudo apt install obs-studio
sudo apt install ffmpeg

Запуск OBS Studio прошел успешно.

Если выдает ошибку графического драйвера системы. Проверьте версию Open CL, должно быть не ниже версии 3.0

Команда проверки версии Open CL:

glxinfo | grep OpenGL

Подключение камеры с андроид смартфона к OBS Studio

установка программы DroidCam Wireless Webcam

  • Способ костыль. в браузере вхожу в http://ipадрессмартфона:4747 и в OBS добавляю источник "Захват окна XComposite" и выбираю нужную программу с которой будет съем изображения.

  • Выбираем источник VLC видео, далее Добавить путь/url и прописываем http://ipадрессмартфона0:4747/mjpegfeed?640x480 и вуяля все готово. Есть задержка изображения со звуком. думаю это будет критично если онлайн трансляция.

у этой аднроид программы есть линукс клиент http://www.dev47apps.com/droidcam/linuxx/

Усть еще одна программа на андроид это IP WebCam - по ощущениям больше возможностей.

Как сделать бегущую строку в трансляции

есть полезное видеоhttps://www.youtube.com/watch?v=xsItRye91jc

Кратко: добавляем в Источниках нужной сцены "Текст", далее в настройках пишем нужный текст или можно также подключить тестовый файл, и потом его можно пополнять и сохранять, также сразу будет меняться текст. удобно делать бегущую строку новостей. Чтобы текст прокручивался, то нужно в свойствах Текста выбрать пункт "Фильтры", и уже в фильтрах добавить "Прокрутка". В параметрах прокрутки можно менять скорость и направление прокрутки.

Как сделать плашечку внизу с именем выступающего и его должность

  • в сцене добавил источник "текст" и далее делаю его видимым если нужно или отключаю.
  • думаю также можно делать красывую плашечку в виде изображения
  • может есть другие более красывые способы..
