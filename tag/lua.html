Компиляция Lua

curl -R -O http://www.lua.org/ftp/lua-5.3.4.tar.gz
tar zxf lua-5.3.4.tar.gz
cd lua-5.3.4
make linux test
  • если возникает ошибка: lua.c:82:10: фатальная ошибка: readline/readline.h: No such file or directory, то нужно установить:
sudo dnf install readline-devel

Установка Lua

sudo make linux install

Install LuaRocks

wget https://luarocks.org/releases/luarocks-2.4.1.tar.gz
tar zxpf luarocks-2.4.1.tar.gz
cd luarocks-2.4.1
./configure
make build
sudo make install
sudo luarocks install luasocket   - в fedora почему-то под sudo не ставится, а вот под root без проблем
