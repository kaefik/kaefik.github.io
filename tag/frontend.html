Полезные плагины VSCode для Frontend

Mon 03 September 2018 | tags: программированиеfrontendjavascriptjsvscode
  • Prettier — JavaScript formatter

Этот плагин позволит нам делать форматирование кода при нажатии shift + alt + f, а также настроить форматирование при сохранении файла. Для этого нужно открыть настройки vs code и установить true для опции editor.formatOnSave.

  • ESLint

Этот плагин будет подсвечивать ошибки и выводить их в нижней панели.

  • Babel ES6/ES7

Этот плагин добавит подсветку кода для jsx и нового синтаксиса js.

  • Simple React Snippets

Плагин, содержащий полезные сниппеты для react, позволяет быстро создавать простой реакт класс и быстро добавлять жизненные методы реакта.

  • NPM Intellisense

Подсказывает название библиотеки при импорте.

  • Path Intellisense

Подсказывает относительный путь модуля при импорте.

  • vscode-styled-components

Подсвечивает CSS-синтаксис при использовании styled-components.

  • Git Lense

Полезный плагин, помогающий просматривать гит-историю, историю изменения файла и сравнение файлов между коммитами.

  • VSCode Great Icons

Добавляет множество иконок к файлам, помогает быстро понять, где свойства, где JS, а где HTML.
