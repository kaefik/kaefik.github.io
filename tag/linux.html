Запуск в docker GUI-приложения в Ubuntu

Sat 17 July 2021 | tags: администрированиеdockerguiubuntulinuxTranslations: ru

Полезные ссылки

  • Как запустить любое ПО с графическим интерфейсом в Docker?
  • ОШИБКА CANNOT OPEN DISPLAY В LINUX

Шаги которые нужно проделать

  1. Создать Dockerfile

dockerfile FROM centos:8 RUN yum install firefox -y CMD ["/usr/bin/firefox"]

или вот такой:

dockerfile FROM ubuntu:20.04 # нужно Configuring tzdata ENV TZ=Europe/Moscow #RUN ln -snf /usr/share/zoneinfo/$TZ /etc/localtime && echo $TZ > /etc/timezone # установка приложения RUN apt update && apt upgrade && apt install firefox -y CMD ["/usr/bin/firefox"]

  1. Создаем образ

bash docker build -t название_образа .

если всё прошло без ошибок то в списке образов будет образ с вашим именем.

  1. Снимаем ограничения доступа к Xorg

для того чтобы приложение могло под root получать доступ. действует до перезагрузки вашего компьютера.

bash sudo xhost +SI:localuser:root

  1. Запускаем докер контейнер

bash docker run --name имя_контейнера -it -e DISPLAY=$DISPLAY --net=host название_образа

если нет переменной окружения DISPLAY:

export DISPLAY=:0
