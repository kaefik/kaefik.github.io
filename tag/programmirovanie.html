Чтобы при сборке докер контейнера не выдавался запрос настрйока time zone, нужно добавить переменную окружения в Dockerfile:

ENV TZ=Europe/Moscow

также при установке некоторых программ все равно идет запрос настройки time zone, помогло добавление DEBIAN_FRONTEND=noninteractive перед командой установки программы:

RUN DEBIAN_FRONTEND=noninteractive apt-get install -y youtube-dl
